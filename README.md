# Git
## 状態の確認
```
git status
```

## コミット
```
git add .
git commit -m "メッセージ"
```
- セーブ

## プッシュ
```
git push
```
- Gitの変更履歴をGitHubに反映

## `.gitignore`
- Gitの管理対象にしないファイルを選択
